# FUNDTAP Dashboard Fork

A beautiful, production-ready dashboard recreation built with React, TypeScript, and Tailwind CSS.

## Features

- 🎨 Modern, responsive design with Tailwind CSS
- 📱 Mobile-first approach with smooth animations
- 🔐 Multi-step verification flow
- 📊 Invoice management system
- 🧭 Intuitive navigation with breadcrumbs
- ✨ Micro-interactions and hover states

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development
- **Tailwind CSS** for styling
- **React Router** for navigation
- **Lucide React** for icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── Header/         # Header component
│   ├── InvoicesSection/ # Invoice management
│   ├── InvoicesTable/  # Data table
│   ├── Navigation/     # Navigation components
│   ├── Pagination/     # Pagination controls
│   └── VerificationItem/ # Verification items
├── pages/              # Page components
│   ├── VerificationPage.tsx
│   └── CompanyDetailsPage.tsx
├── icons/              # Custom icons and logos
└── App.tsx             # Main application component
```
